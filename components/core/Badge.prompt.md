Status pill / label chip for categories, states, and counts.

```jsx
<Badge variant="success" dot>Active</Badge>
<Badge variant="warning" dot>Pending review</Badge>
<Badge variant="danger">Failed</Badge>
<Badge variant="primary">New</Badge>
<Badge variant="blue">Beta</Badge>
<Badge variant="outline">Draft</Badge>
```

**Variants**: default · primary · blue · success · warning · danger · outline

Use `dot` for live status indicators (active, online, pending). Use without dot for category labels, counts, or feature flags.

Keep label text short — 1–2 words max.
